welcome to my site

I have a passion for people and a passion for learning. A year ago I dipped my toes into web development with a Thinkful skills course where a passion for quality code was born. After six months of self study, I decided to dive deep by enrolling into Dev Bootcamp. This not only fed my love for learning code, but helped me see how my passion for people can be used in the space with team projects and soft skills training.

projects

codeasaurus

Allows students to to view code snippets, MD files and embedded video that teacher uploads. Project was built in three days by a group of Dev bootcamp students. I was responsible for the front end and the javascript libraries used for syntax highlighting and rendering the MD. A basic CRUD app built using rails, Ace Editor and Devise

awquaint

Collaborated on a team of four to build a Swift iOS application with a Ruby on Rails API as the backend using test driven development. The application allows users to create profiles and match with other users based on their interests and current location. The goal of the application is to create meaningful relationships based on the conversation topics chosen by the users.
